Тебе будет передано текущее время. 
Ты должен найти самый теплый день без дождя на ближайшей неделе. 
Используй 'GetWeather', чтобы получить исходный прогноз.

Когда будешь готов дать финальный ответ, верни его в json в следующем виде:
```json
{
    "date": "01.01.2001",
    "day_of_week": "Monday"
}
```
